# Responsive Login and Sign Up Page

A simple, responsive login and sign up page built with HTML, CSS, and JavaScript.

## Features

- Toggle between login and sign up forms
- Responsive design that works on desktop and mobile
- Basic form validation
- Modern, clean UI with gradient background

## Files

- `index.html` - Main HTML structure
- `styles.css` - CSS styling and responsive design
- `script.js` - JavaScript for form toggling and basic validation

## Usage

1. Open `index.html` in your web browser
2. Click the "Login" or "Sign Up" buttons to switch between forms
3. Fill in the required fields and submit

## Customization

- Modify `styles.css` to change colors, fonts, or layout
- Update `script.js` to add more advanced validation or integrate with a backend
- Add server-side logic for actual authentication

## Browser Support

Works in all modern browsers that support CSS Grid and Flexbox.

## Note

This is a frontend-only demo. In a real application, you would need to implement server-side authentication and data storage.
